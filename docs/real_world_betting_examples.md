## ğŸ“Š Bet Type Definitions

### 1. **Moneyline** (ML)
Straight-up winner bet. No point spread.
- **Favorite**: Negative odds (e.g., -150 means bet $150 to win $100)
- **Underdog**: Positive odds (e.g., +130 means bet $100 to win $130)

### 2. **Point Spread / Puck Line**
Handicap betting - favorite must win by X points, underdog can lose by less than X.
- **Example**: Lakers -4 means Lakers must win by 5+ points
- **NHL uses 1.5 goal spread** (called puck line)

### 3. **Over/Under (Totals)**
Bet on combined score of both teams.
- **Over**: Total score > line
- **Under**: Total score < line

### 4. **Parlays**
Combine multiple bets - all must win for payout.
- Higher risk, higher reward
- Example: 3-leg parlay at +600 means $100 bet wins $600

### 5. **Prop Bets**
Bet on specific events within game.
- Player props: Points, rebounds, goals, saves
- Team props: First to score, halftime leader
- Game props: Longest touchdown, etc.

**Rare but possible:**
- **Book A**: Rams -3.5 (-105)
- **Book B**: Bears +4.5 (-110)

**Bet both:**
- $105 on Rams -3.5
- $110 on Bears +4.5

- **Guaranteed win**: If Rams win by exactly 4, BOTH bets win
- Otherwise one bet wins (~$100), net loss minimal

**Scenario**: You bet $100 on Bills to win Super Bowl at +1200 (win $1,200).

- **Guaranteed profit either way**

## ğŸ”¢ Odds Conversion Reference

### American to Probability
```python
def american_to_probability(odds):
    if odds < 0:  # Favorite
        return abs(odds) / (abs(odds) + 100)
    else:  # Underdog
        return 100 / (odds + 100)

# Examples:
# -250 â†’ 71.4%
# +215 â†’ 31.7%
# -110 â†’ 52.4%
```

### Probability to American
```python
def probability_to_american(prob):
    if prob > 0.5:  # Favorite
        return -1 * (prob / (1 - prob)) * 100
    else:  # Underdog
        return ((1 - prob) / prob) * 100

# Examples:
# 65% â†’ -186
# 35% â†’ +186
# 50% â†’ +100
```

### Implied Probability vs. True Probability
```
Book offers: Lakers -4 at -110 (52.4% implied)
Your model says: Lakers 60% to cover -4
Edge = 60% - 52.4% = 7.6% (+EV bet!)
```
